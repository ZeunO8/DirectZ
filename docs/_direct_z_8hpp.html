DZ_EXPORT EventInterface * init ( const WindowCreateInfo & window_info)

Initializes the runtime environment.

This function should be implemented by the platform-specific runtime (e.g., PC, Android, iOS). It typically creates the main window, caches it for later use, and sets up runtime resources such as shaders and images.

// Example implementation:
WINDOW* cached_window = 0;
DZ_EXPORT EventInterface* init(const WindowCreateInfo& window_info)
{
cached_window = window_create(window_info);
// shader & image setup
return window_get_event_interface(cached_window);
}
