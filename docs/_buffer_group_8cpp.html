◆ buffer_group_get_buffer_element_view()

ReflectedStructView buffer_group_get_buffer_element_view ( BufferGroup * buffer_group,
const std::string & buffer_name,
uint32_t index )

Gets a reflected view of a specific struct element within a shader buffer. This view allows updating individual members of the struct by name, handling memory layout differences (padding, alignment) automatically.

This function is intended for buffers whose element_type is a struct (e.g., UBOs, SSBOs of structs).

Parameters
shaderThe shader object.
buffer_nameThe GLSL variable name of the buffer (e.g., "ubo_scene", "particles").
indexThe 0-based index of the element to view (for SSBOs). For UBOs, this is usually 0.
Returns
A ReflectedStructView object.
Exceptions
std::runtime_errorif the buffer/element is not found, data_ptr is null, index is out of bounds, or the element type is not a struct.
